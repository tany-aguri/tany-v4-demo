# tany農園 v13 レビュー版

奥様が描いた夫婦のイラストをHomeに正式反映した、GitHub Pages用UIです。

## v13の主な更新
- Homeに「Tanimas blog」の手描き夫婦イラストを正式採用
- 黒い余白やスマートフォン画面部分を取り除き、Web表示向けに最適化
- イラストを単なる飾りではなく「tany農園の原点」として配置
- v11で整理した役割分担を維持
  - Home：今日の畑・今日の楽しみ・記録への入口
  - 栽培：現在地・残り工程・採種を含む栽培カレンダー
  - 記録：保存済み記録の閲覧と追加
  - 図鑑：作物ごとの育った物語
- AI設定は設定画面下部の補助機能として控えめに配置

## GitHub Pagesへの反映
このフォルダ内のファイルと `assets` フォルダを、リポジトリ直下へまとめてアップロードしてください。既存ファイルは上書きで問題ありません。

記録と写真はブラウザのlocalStorageに保存され、外部送信されません。
